Step 1: Add below script to your <Head> tag

<script src="https://shrihari-lib.netlify.app/lineage.min.js"/>

Step 2: Prepare the family Data


var jsonData = [
{ "id": 1, "pid": 2, "gender": "male", "title": "Title", "name": "Father", "photo": "", "addr": "NY", "cn":"us" },
{ "id": 2, "pid": 1, "gender": "female", "title": "Title", "name": "Mother", "photo": "", "adddr": "NY" , "cn":"us"},
{ "id": 3, "pid": 4, "mid": 2, "fid": 1, "gender": "female", "title": "Title", "name": "Daughter",
    "photo": "//unsplash.it/80/80", "addr": "USA" , "cn":"us"},
{ "id": 4, "pid": 3, "gender": "male", "title": "Title", "name": "Son-in-Law","photo": "", "addr": "", "cn":"ca" },
];

Here are the details of Data

    
    id  identification - unique - required
    gender  male / female - required
    name  name of family member - required
    title  family name - optional
    photo  photo url (local / web) - optional
    addr  address
    cn  country code, refer to this link to find your country code ISO 3166 Country Code. 
                        Note: No image will be shown if you use wrong code.
    pid  partner id  - (optional) *
    mid  mother id - (optional) *
    fid  father id - (optional) *
    * keep note one of these are required
        partner(pid) or parents(mid, fid) to generate the family Tree

Step 3 : Add before </Body>

    
<body>
    <div id="divFamily"></div>
    <script>
        var params = {
            data: jsonData, /*Local variable or file path*/
            search: true, //false
            container: "divFamily",
            template: "circle" // "rounded" // "raised" // "tilted"
        };
        var tree = new Lineage(params);
        tree.load();
    </script>
</body>

Features

Search

select and see the decendends (if any)

Click on Member

To see the bloodline (if any)


Lineage
